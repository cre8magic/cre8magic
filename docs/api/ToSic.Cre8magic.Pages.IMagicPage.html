Magic Pages are smart wrappers around Oqtane Pages. They offer many benefits such as:

  1. Read-only properties (no accidental writing properties with unexpected side effects)
  2. Improved naming, e.g. .Id instead of .PageId
  3. Calculated properties such as Target (which becomes "_blank" for external links or null for normal links)
  4. Corrected values - e.g. the Link property will be javascript:void(0) if the page is not clickable
  5. Navigation properties such as Parent and Breadcrumb

MenuLevel

Level in the menu, starting from 1. As different menus may start at other depths, this is not always the same as the Oqtane level.

Note that when building menus or breadcrumbs, this menu-level is recalculated to fit the current menu. So if the menu itself starts at another level (e.g. showing only all pages under Products), then these pages would declare they are MenuLevel=1
