The cre8magic Analytics helps you integrate Google Analytics using Google Tag Manager and Track Page Views.

Tip

A simple top-level menu can be added just like this:

<MagicAnalytics Settings='new() { GtmId = "GTM-T8W5TBL", PageViewTrack = true }'></MagicAnalytics>

...or if you have a theme.json file, all you need is:

<MagicMenu Name="MainMenu"/>

Test

<MagicAnalytics Settings='new() { GtmId = "GTM-T8W5TBL", PageViewTrack = true }'></MagicAnalytics>
